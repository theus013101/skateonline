## 🎯 Objetivo do Software
Sistema desktop para cadastro e consulta de produtos de skate, integrado ao banco de dados MySQL.

## ⚙️ Funcionalidades
- Cadastro de produtos (nome, valor, quantidade)
- Consulta e listagem de produtos
- Interface gráfica em Swing
- Integração com banco de dados MySQL
